# Campaign Template

Copy this folder to start a campaign:

```sh
cp -R campaigns/_template campaigns/my-campaign-name
```

Then edit `campaign.md` and begin adding logs, characters, summaries, lore, and Avrae notes.

## What Goes Where

- `00-inbox/`: unsorted pasted text, quick notes, rough imports
- `01-characters/`: campaign PCs, NPC companions, rivals, recurring NPCs
- `02-chat-logs/`: raw Discord and Avrae logs
- `03-session-summaries/`: cleaned summaries by session, scene, or date
- `04-lore/`: locations, factions, quests, clues, magic items, timelines
- `05-avrae/`: aliases, snippets, command examples, character sheet links
- `06-prompts/`: prompts tailored to this campaign
- `07-private-notes/`: spoiler-sensitive or private notes
