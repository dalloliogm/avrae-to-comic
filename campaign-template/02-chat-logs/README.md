# Chat Logs

Keep raw Discord and Avrae logs here.

Suggested format:

```text
YYYY-MM-DD-channel-or-scene.md
```

Examples:

```text
2026-05-15-tavern-introduction.md
2026-05-16-avrae-combat-goblin-ambush.md
```

Try to preserve:

- speaker names
- timestamps, if available
- Avrae rolls and embeds
- channel names
- scene boundaries

It is fine if early imports are messy. Put them here anyway; cleanup can happen later.
